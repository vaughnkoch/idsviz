# IDS Seattle - Machine Learning / Dataviz Talk

## Requirements
- python 3.7
- pip
- virtualenv
- mkvirtualenv

## Installation

```# Enter this in new Terminal sessions with 'workon idsviz'
mkvirtualenv idsviz
pip install -r requirements.txt

# Open Jupyter - once it loads, pick Viz.ipynb
jupyter notebook
```
